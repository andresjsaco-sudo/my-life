🔬 Final Degree Project — In Progress

Early Corrosion Detection System for Steel — IoT + Computer Vision

An interdisciplinary IoT system for detecting early-stage corrosion on steel plates in industrial environments. Developed in collaboration with Corpoacero (Barranquilla) and teams from mechanical and electronic engineering. A camera transmits daily images via LTE to a cloud backend where a custom-trained ML model identifies corrosion indicators invisible to the naked eye. Results are visualized on a web app with interactive plant maps showing per-plate status in real time.

  • IoT / LTE
  • Custom ML Model
  • Industrial Application
  • Web Dashboard

Communications Lab — Universidad del Norte

IoT Occupancy Monitoring System with 3D Point Cloud Visualization

End-to-end IoT system for real-time person detection and spatial localization in university classrooms. A Raspberry Pi 4 runs YOLOv8 Nano on-device (edge AI) at 6–10 FPS, transmitting detection coordinates via LTE (SIMCOM A7670SA) over MQTT/TLS to AWS IoT Core. Data flows through Lambda and DynamoDB into a Next.js dashboard and a Potree 3D point cloud viewer, displaying occupancy in near real-time with ~300ms latency.

  • YOLOv8 Nano
  • Edge AI
  • Raspberry Pi 4
  • LTE / MQTT
  • AWS IoT Core
  • AWS Lambda
  • DynamoDB
  • Potree 3D
  • Next.js
